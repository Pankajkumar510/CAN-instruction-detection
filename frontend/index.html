01 · Abstract

Why CAN Bus Security Matters

Modern vehicles contain 70+ ECUs communicating over Controller Area Network (CAN) — a protocol designed without authentication. We address this glaring attack surface.

⚠

The Threat Landscape

CAN bus lacks authentication mechanisms, making every connected vehicle susceptible to DoS, Fuzzy, Replay, and Spoofing attacks. A compromised ECU can seize steering, brakes, or engine control with potentially fatal consequences.

⬡

Our Contribution

We propose TinyCNNCANNet — a family of three lightweight 1D-CNN architectures optimized for edge deployment. With only ~13K parameters, they run inference in microseconds on resource-constrained ECU hardware.

◎

SynCAN2025 Dataset

We introduce SynCAN2025 — a new synthetic CAN intrusion detection benchmark generated via realistic vehicle simulation with controlled attack injection, enabling rigorous generalization evaluation across attack scenarios.

Key Contributions

①
Three Lightweight CNN Architectures

TinyCNNCANNet with ~13K parameters and two efficient baseline models, all achieving competitive detection accuracy.

②
SynCAN2025 Benchmark

A new synthetic CAN intrusion detection dataset generated through realistic simulation and controlled attack injection.

③
Extensive Multi-Dataset Evaluation

Experiments on four diverse datasets analyzing detection performance, computational efficiency, and attack scenario coverage.
